동기적
먼저 시작한 일이 종료되어야 다음 일을 처리하는 것

비동기적
병렬적으로 동시에 여러 가지 일을 처리하는 것

callback
비동기적으로 사용하기 위해 정의해둔 함수가 처리되면 실행한다.
사용법은 매개변수로 callback을 쓰고 안에서 callback을 실행한다.

익명함수
이름이 없는 함수 - 변수에다 저장할 수 있다 == 함수는 값이다.

패키지
소프트웨어를 부르는 여러가지 표현 중에 하나.

Package Manager
소프트웨어들을 관리해주는 역할

request 컴퓨터가 요청할 때 웹브라우저가 보낸 정보
response 응답할 때 우리가 웹브라우저 한테 전송할 정보

리다이렉션
사용자가 글을 쓰고 자기가 쓴 글이 잘 전송 됐는지 확인하는 기능
301 - 일시적으로 주소 변경?

리펙토링
동작 방법은 똑같이 유지하면서 내부의 코드를 더 효율적으로 바꾸는 행위

캐싱
사용자들이 어떤 페이지를 크릭하면 빠르게 글로 갈 수 있도록
미리 웹페이지를 다운로드 받는 것

객체(object) 리터럴 중괄호
순서가 없는 정보를 저장하기에 최적이다.
이름으로 식별자를 줄 수 있다.

배열(array) 리터럴 대괄호
정보를 정리정돈할 때 순서에 따라서 정리정돈한다.
각각의 엘리먼트(정보)들은 고유한 식별자가 있다.

함수
변수의 값으로 지정할 수 있고 배열의 원소로 존재할 수 있고 객체의
원소(property)로도 올 수 있다.

모듈
가장 큰 정리정돈 도구? 객체나 함수 등을 모아 놓은 것
파일로 쪼개서 외부로 독립시킬 수 있다.

보안
웹의 id값을 수정하여 접속할 수 있는 경우 페이지의 중요한 정보
또는 컴퓨터의 정보들이 유출 될 수 있어서 보안이 중요하다.



문법 관련

readFileSync("파일 경로(이름)",'utf-8);
동기적으로 파일을 읽는다. 리턴 값이 있다.

readFile("파일 경로(이름)",'utf-8',function(err,result(콜백){
	//에러가 있다면 에러를 인자로 제공 아니면 result 제공
});
비동기적으로 파일을 읽는다. 리턴 값이 없다.(콜백함수를 3번쨰 인자로 준다.)
먼저 문장이 쓰여서 출력하더라도 밑의 구문이 빠르면 밑을 먼저 시작한다.
= 콜백 함수(3번째 인자)를 이용해 함수의 작업이 끝나면 콜백함수를 실행시킨다.

<textarea>
텍스트를 여러 줄 적을 수 있다.

<form action = "주소">
폼 안에 있는 각각의 컨트롤들의 사용자가 입력한 정보를 서브밋 버튼을 눌렀을 때
액션 속성이 가리키는 서버로 쿼리 스트링의 형태로 데이터를 전송하는 기능이다.

method = "post"
쿼리 스트링을 감출 때 사용
서버에 데이터를 수정 삭제 생성과 같은걸 할 때 반드시 사용해야한다.

객체
var 객체이름 = {
    '키':'value'
}
객체 for문
for(var 변수 in 객체){
	반복문
}
변수를 출력하면 객체의 키값이 차례대로 나온다.
객체[변수]를 하면 value가 나온다.

module.exports
모듈 바깥에서 사용할 수 있도록 추출한다.

path.parse(내용).base

sanitized
